
# Simple Discord Bot

simple discord bot to find out minecraft server status


## How to run

install Discord.js and node-fetch

```bash
  npm install discord.js
```

install node-fetch

```bash
  npm install node-fetch@2
```

and run

```bash
  node index.js
```
